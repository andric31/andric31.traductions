Repère rapide

Titre = nom de l’article visible sur le blog. Slug = nom du fichier et du lien web. Date affichée = texte vu par les visiteurs. Date ISO = format technique pour trier proprement. Catégorie = petit label visible sur la carte et dans l’article.
