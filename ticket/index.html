Envoyer un ticket

Utilise ce formulaire pour envoyer une demande simple à un administrateur : problème sur le site, suggestion d’amélioration, signalement ou demande d’accès privé.

Information importante Le ticket sera visible uniquement par un administrateur. Ce formulaire ne crée pas de conversation : aucune réponse ne sera envoyée. Pour une question qui demande un échange ou une réponse, utilise plutôt la page Messages.

Pour une vraie conversation, les messages sont plus adaptés. Le ticket sert surtout à envoyer une information ou une demande sans suivi direct.

Indique dans le message le pseudo et le mot de passe souhaité.
