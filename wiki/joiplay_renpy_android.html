📱 Lancer un jeu Ren’Py Windows (et sa traduction) avec JoiPlay sur Android

Ce guide explique comment jouer à un jeu Ren’Py (version Windows) sur Android avec JoiPlay, et comment y intégrer ta traduction ✅

🧩

Objectif

Tu vas : télécharger le jeu + la traduction, copier la traduction dans le dossier du jeu, puis ajouter le .exe dans JoiPlay pour lancer le jeu traduit 🎉

✅ Important : prends toujours une traduction qui correspond à la même version que ton jeu.

1️⃣ Télécharger le jeu Ren'Py et sa traduction correspondante

Commence par télécharger le jeu Ren'Py en version Windows et la traduction adaptée à sa version. Une fois les deux fichiers téléchargés sur ton appareil Android, extrais-les.

💡 Tu obtiendras deux dossiers distincts : un pour le jeu, un pour la traduction.
💡 Tu peux ensuite supprimer les archives .zip et .rar afin de gagner de l’espace sur Android.
⚠️ Il est recommandé de placer tous tes jeux dans un dossier dédié que tu ne déplaceras pas, même après l’installation dans JoiPlay.

2️⃣ Copier les fichiers de la traduction

Ouvre le dossier de la traduction, sélectionne tous ses fichiers et dossiers, puis copie-les dans le dossier du jeu.

✅ Quand Android te demande si tu veux remplacer les fichiers existants, choisis Remplacer.
🟦 Et voilà, le jeu et la traduction sont prêts ! Ajoute ensuite le jeu (avec la traduction) dans JoiPlay 🎉

Si tu n’utilises pas JoiPlay, tu peux t’arrêter ici ✅
Sinon, passons à l’étape suivante : l’installation, l’ajout du jeu et son utilisation. 👇

3️⃣ Télécharger JoiPlay

Pour une meilleure expérience, il est recommandé de télécharger la dernière version de JoiPlay et les plugins depuis le Patreon du créateur.

  • Patreon : JoiPlay (page officielle)
  • Version utilisée pour ce tutoriel (1.20.611) : Post Patreon 1.20.611
  • Plugins mentionnés : Ren'Py 8.4.1 Plugin 1.00.10 et Ren'Py 7.7.1 Plugin 1.00.60
💡 JoiPlay permet d’exécuter des jeux Ren'Py Windows directement sur Android, sans avoir besoin d’un PC.

4️⃣ Installation de JoiPlay

💡 Certains systèmes Android bloquent l’installation d’APK provenant de sources extérieures à Google Play.
Comme nous utilisons des versions hors Play Store, tu dois autoriser l’installation d’applications inconnues dans :
Paramètres → Sécurité et confidentialité → Autres paramètres de sécurité → Installation d’applis inconnues

Pour plus d’informations, consulte le manuel ou l’aide du système Android que tu utilises.

Ensuite, lance le fichier APK de JoiPlay et installe les plugins sur ton appareil Android.

5️⃣ Lancement de JoiPlay

Une fois installé, ouvre JoiPlay depuis ton écran d’accueil Android.

💡 L’application te demande des autorisations (comme l’accès au stockage) : accepte-les, sinon le jeu ne pourra pas se lancer !

6️⃣ Ajouter le jeu dans JoiPlay

Depuis l’écran d’accueil de JoiPlay, appuie sur le bouton ➕ puis sélectionne Add Game. Ensuite, choisis le fichier .exe du jeu Ren'Py dans le dossier du jeu (avec la traduction que tu as copiée, hein 😉).

Pour finir l’ajout

Une fenêtre s’ouvre et te demande de renseigner plusieurs champs :

  • Game Name → le nom du jeu (tu peux mettre ce que tu veux)
  • Version → la version du jeu (tu peux mettre ce que tu veux)
  • Icon → ajoute une image si tu veux personnaliser ton jeu
💡 Conseil : mets le nom + la version dans “Game Name” (ex : NomDuJeu v0.12.0) pour t’y retrouver plus tard.

7️⃣ Lancer le jeu traduit

Une fois ton jeu ajouté dans JoiPlay, il apparaîtra sur l’écran d’accueil de l’application.

Pour le lancer, appuie simplement sur son icône ✅
Le jeu s’ouvrira alors directement avec la traduction en place !

💡 Avant que le jeu se lance, plusieurs autorisations vont t’être demandées. Pour jouer, il faut les accepter.
🎮 Bon jeu !

🧰 Dépannage

❌ Le jeu ne démarre pas (écran noir / retour au menu JoiPlay)

  • Vérifie que les plugins Ren’Py sont bien installés (Ren’Py 7/8 selon le jeu).
  • Vérifie que tu as bien choisi le .exe du jeu (pas un fichier au mauvais endroit).
  • Évite de déplacer le dossier du jeu après l’avoir ajouté dans JoiPlay.

❌ Le jeu n’est pas traduit

  • Vérifie que les fichiers de traduction ont bien été copiés dans le bon dossier du jeu.
  • Vérifie la structure : game/tl/fr_zenpy_DeepL_andric31/...
  • Vérifie que la traduction correspond à la même version que le jeu.
⚠️ Si tu as un doute, supprime le dossier de traduction (game/tl/fr_zenpy_DeepL_andric31) et recopie la traduction proprement.
