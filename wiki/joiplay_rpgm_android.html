📱 Traduction RPGM sur Android + lancement avec JoiPlay

Ce guide explique comment installer une traduction pour un jeu RPG Maker (RPGM, VX, MV, MZ) sur Android, puis le lancer via JoiPlay ✅

🧩

Objectif

Tu vas : télécharger le jeu + la traduction, copier la traduction dans le dossier du jeu, puis ajouter le jeu dans JoiPlay pour le lancer en français 🎉

✅ Important : prends toujours une traduction qui correspond à la même version que ton jeu.

1️⃣ Télécharger le jeu RPGM et sa traduction correspondante

Commence par télécharger le jeu RPGM et la traduction adaptée à sa version. Une fois les deux fichiers téléchargés sur ton appareil Android, extrais-les.

💡 Tu obtiendras deux dossiers distincts : un pour le jeu, un pour la traduction.

2️⃣ Copier les fichiers de la traduction

Assure-toi que les deux dossiers sont bien visibles sur ton appareil Android. Ouvre ensuite le dossier de la traduction, sélectionne tous ses fichiers et dossiers, puis copie-les dans le dossier du jeu.

✅ Quand ton téléphone te demande si tu veux remplacer les fichiers existants, choisis Remplacer.
🟦 Et voilà, la traduction est prête ! Tu peux maintenant profiter du jeu en français 🎉

Si tu utilises déjà JoiPlay, la suite ne devrait pas trop t’intéresser — alors bon jeu, et merci de m’avoir lu.
Mais pour celles et ceux qui ont encore besoin d’un petit coup de main, passons à l’étape suivante : l’installation et l’utilisation de JoiPlay. 👇

3️⃣ Télécharger JoiPlay

Télécharge l’application JoiPlay sur le Play Store ou directement sur la page officielle de JoiPlay.

💡 JoiPlay permet d’exécuter les jeux RPG Maker (RPGM, VX, MV, MZ) directement sur Android, sans avoir besoin d’un PC.
💡 Pense aussi à télécharger le plugin RPG Maker.

4️⃣ Installation de JoiPlay et du plugin RPG Maker

💡 Certains systèmes Android bloquent l’installation d’APK provenant de sources extérieures à Google Play.
Si tu as téléchargé les fichiers directement (hors Play Store), tu dois autoriser l’installation d’applications inconnues dans :
Paramètres → Sécurité et confidentialité → Autres paramètres de sécurité → Installation d’applis inconnues

Pour plus d’informations, consulte le manuel ou l’aide du système Android que tu utilises.

Une fois cette option activée, lance les fichiers APK correspondants (JoiPlay et le plugin RPG Maker) pour les installer sur ton appareil Android.

5️⃣ Lancement de JoiPlay

Une fois installé, ouvre JoiPlay depuis ton écran d’accueil Android.

💡 L’application te demande des autorisations (comme l’accès au stockage) : accepte-les, sinon le jeu ne pourra pas se lancer !

6️⃣ Ajouter le jeu dans JoiPlay

Depuis l’écran d’accueil de JoiPlay, appuie sur le bouton ➕ puis sélectionne Add Game. Ensuite, choisis le fichier index.html dans le dossier de ton jeu (avec la traduction que tu as mise, hein 😉).

Pour finir l’ajout

Une fenêtre s’ouvre et te demande de renseigner plusieurs champs :

  • Game Name → le nom du jeu (tu peux mettre ce que tu veux)
  • Version → la version du jeu (tu peux mettre ce que tu veux)
  • Icon → ajoute une image si tu veux personnaliser ton jeu
💡 Conseil : mets le nom + la version dans “Game Name” (ex : NomDuJeu v1.2.3) pour t’y retrouver plus tard.

7️⃣ Lancer le jeu traduit

Une fois ton jeu ajouté dans JoiPlay, il apparaîtra sur l’écran d’accueil de l’application.

Pour le lancer, appuie simplement sur son icône ✅
Le jeu s’ouvrira alors directement avec la traduction en place !

🎮 Bon jeu !

🧰 Dépannage

❌ Le jeu ne démarre pas / écran noir

  • Assure-toi que le plugin RPG Maker est bien installé.
  • Évite de déplacer le dossier du jeu après l’avoir ajouté dans JoiPlay.

❌ Le jeu n’est pas traduit

  • Vérifie que les fichiers de traduction ont bien été copiés dans le bon dossier du jeu.
  • Si on te demande “Remplacer”, il faut accepter ✅
⚠️ Si tu as un doute, recopie la traduction proprement (et remplace les fichiers).
