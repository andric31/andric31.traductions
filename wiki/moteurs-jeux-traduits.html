🧠 Moteurs de jeux (que je traduis)

Les principaux moteurs que je rencontre dans mes projets de traduction.

🐍 Ren’Py

🗂️ Formats principaux : .rpy .rpyc
🌐 Site officiel : renpy.org

Ren’Py est un moteur open source spécialisé dans les Visual Novels. 📖💖 Il utilise le langage Python, ce qui le rend puissant et personnalisable. 🐍 C’est l’un des moteurs les plus répandus dans le domaine des jeux narratifs.

🏰 RPG Maker MV (RPGM)

🗂️ Formats principaux : .json .js
🌐 Site officiel : rpgmakerweb.com

RPG Maker est spécialisé dans les jeux de rôle 2D classiques. ⚔️🗺️ Il s’appuie sur JavaScript pour la personnalisation et utilise des fichiers JSON pour stocker les données.

🎮 Unity

🗂️ Formats principaux : .json .xml .csv .txt
🌐 Site officiel : unity.com

Moteur très répandu pour la 2D et la 3D. 🌍✨ Les fichiers peuvent contenir dialogues, configurations et données de jeu.

👾 🌀 Godot

🗂️ Formats principaux : .pck (.tscn .tres .dtl .gd)
🌐 Site officiel : godotengine.org

Godot est un moteur open source très flexible, utilisé en 2D comme en 3D. 🖼️🧊 Il utilise son propre langage : GDScript.

🦊 🦖 TyranoBuilder (TyranoScript)

🗂️ Formats principaux : .ks .json
🌐 Site officiel : tyranobuilder.com

Tyrano est un moteur japonais spécialisé dans les Visual Novels. 🎌💬 Il repose sur une syntaxe KS simple, personnalisable via JavaScript.

🐟🐟 KiriKiri / KAG

🗂️ Formats principaux : .xp3 (.ks .tjs)
🌐 Site officiel : krkrz.github.io

Moteur japonais très répandu pour les Visual Novels. 🗾💬 Utilise KAG Script pour les dialogues et TJS (proche de JavaScript) pour la logique.

🧾🎨 Visual Novel Maker (VNM)

🗂️ Formats principaux : .json .xml
🌐 Site officiel : visualnovelmaker.com

Développé par Kadokawa (RPG Maker) et distribué par Degica. 🎭💬 Moteur dédié aux Visual Novels, avec personnalisation via JavaScript.

⚡💻 Electron

🗂️ Formats principaux : .json .js .html .css (.asar)
🌐 Site officiel : electronjs.org

Framework open source permettant de créer des applications desktop avec des technologies web. 🌐💻 Certains jeux utilisent Electron comme base.

🌐 NW.js (Node-Webkit)

🗂️ Formats principaux : .nw .json .js
🌐 Site officiel : nwjs.io

Combine Node.js et Chromium pour exécuter des jeux ou applis comme un programme PC. 💻 Permet de transformer un projet HTML5/JavaScript en application autonome.
