📱 Mettre une traduction Ren'Py sur Android (avec un ordinateur)

Ce guide explique comment installer une traduction Ren’Py sur Android en copiant les fichiers depuis un ordinateur (via USB) ✅

🧩

Objectif

Tu vas : installer l’APK, lancer le jeu une fois (pour créer les dossiers), puis copier la traduction dans le bon emplacement depuis ton PC.

✅ Astuce : garde toujours la même version du jeu et la même version de traduction.

1️⃣ Installer XXX.apk sur Android

Avant tout, assure-toi de télécharger l’APK de ton jeu depuis une source fiable.

💡 Certains systèmes Android bloquent l'installation d'APK provenant de sources extérieures à Google Play. Active le mode d'installation d'applications inconnues dans :
Paramètres → Sécurité et confidentialité → Autres paramètres de sécurité → Installation d’applis inconnues

Pour plus d'informations, consulte le mode d'emploi du système Android utilisé.

2️⃣ Lancer le jeu une fois et quitter

Lance le jeu une première fois puis quitte. Ça permet la mise en place de certains dossiers.

💡 Si le jeu demande une autorisation (accès au stockage, par exemple), il faut l'autoriser, sinon pas de sauvegarde !
Ferme bien le jeu après cette première ouverture, même en arrière-plan, hein 😉

3️⃣ Connecter le système Android à un ordinateur

⚠️ Attention à Android 15 : l'accès n'est plus possible par USB ! 🔌🚫

Branche ton appareil Android à l’ordinateur avec un câble USB.

💡 Sélectionne le mode « Transfert de fichiers » lorsque l’invite apparaît sur ton téléphone.
Cela permettra à l’ordinateur d’accéder au stockage de l’appareil.

Pour plus d'informations, consulte le mode d'emploi du système Android utilisé.

4️⃣ Accès au dossier du jeu Android sur l'ordinateur

⚠️ Attention à Android 15 : l'accès n'est plus possible par USB ! 🔌🚫

Une fois ton appareil connecté en mode transfert de fichiers, explore son stockage pour trouver le dossier du jeu.

Par défaut, le jeu se trouve dans :

Ce PC\...\Stockage interne\Android\data

Si le jeu est sur carte SD :

Ce PC\...\Carte SD\Android\data

🔎 Cas particuliers

  • Si c'est un apk non officiel (modifié ou fourni par un tiers), la structure de dossiers peut être différente. Les fichiers peuvent aussi être ailleurs (ex : dossier Documents).
  • ⚠️ Le dossier peut se nommer différemment du jeu.
    Pour le trouver plus facilement : trie par « Modifié le » et repère le dernier dossier créé/modifié.

5️⃣ Ajouter la traduction

Pour appliquer la traduction, copie les fichiers de traduction dans le bon emplacement.

✅ Copier la traduction dans le dossier files : 
...\files\game\tl\[dossier_de_traduction]\fichiers_de_traduction.rpy
⚠️ Méthodes alternatives selon le traducteur :
Suivant le traducteur, la méthode peut être différente. 
# soit
Copier le dossier "game" de la traduction dans "files"
(dans "game", le dossier "tl" doit être présent)

# soit
Créer un dossier "game" puis copier ou créer le dossier "tl" dans "game"

# et soit
Si pas de dossier "tl" :
Créer le dossier "tl" dans "game"
Puis copier le dossier de traduction :
"fr" ou "french" ou "fr_XXX_andric31" dans "tl"

6️⃣ Lancer le jeu sur Android

Ouvre le jeu normalement.

💡 Au besoin, va dans les paramètres du jeu pour changer la langue.
🎮 Bon jeu !
