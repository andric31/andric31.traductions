🎮 Appliquer la traduction d'un jeu Godot (.pck)

Méthode automatique et manuelle pour les jeux utilisant un fichier .pck

Certains jeux Godot utilisent un fichier .pck pour regrouper leurs ressources (scripts, images, sons, etc.) en un seul fichier.

Cela simplifie la distribution et le chargement. Pour appliquer une traduction, il faut donc modifier ou remplacer ce fichier afin d’y intégrer les textes traduits.

📦 Dans l’archive de traduction du jeu, deux méthodes sont proposées :

🎯 Méthode Automatique (Plug & Play)

La plus simple : un exécutable applique directement la traduction sans rien installer de plus. Tout est déjà inclus !

  • Placez puis décompressez l’archive de traduction dans le dossier principal du jeu (celui qui contient l’exécutable et le fichier .pck).
  • Double-cliquez sur xxx_fr_patch_traduction.exe.
  • Une fenêtre s’ouvrira.

Cliquez sur "Appliquer" pour lancer le processus.

  • Injection : les fichiers traduits sont intégrés dans le .pck.
  • Finalisation : le nouveau .pck patché remplace l’original.

Une fois le message "Patch appliqué avec succès." affiché, cliquez sur OK.

Vous pouvez ensuite supprimer l'exécutable et l'archive.

Lancez le jeu normalement et profitez de la traduction !

🔧 Méthode Manuelle (Avancé)

Si vous préférez gérer vous-même la traduction, j’ai inclus les fichiers traduits dans l’arborescence correcte pour une intégration manuelle.

⚠️ Cette méthode nécessite de manipuler le fichier .pck avec des outils adaptés et demande quelques connaissances techniques.
