🧪 Vérifier une traduction Ren'Py

Cet outil vérifie si une traduction Ren'Py est bien placée dans le dossier du jeu.
Les fichiers sont lus uniquement par ton navigateur : rien n’est envoyé sur Internet.

📁 Choisir le dossier contenant le .exe du jeu

Choisis le dossier principal du jeu, celui où se trouve le fichier .exe. L’outil vérifie uniquement la structure utile, sans afficher la liste des fichiers.

✅ Conseil : dans la fenêtre Windows de sélection du dossier, les fichiers comme .exe ne sont pas affichés, c’est normal. Sélectionne le dossier qui contient game, lib et renpy, puis clique sur Importer : le nom du .exe détecté s’affichera dans le résultat.

ℹ️ Ce que l’outil vérifie

  • Présence d’un fichier .exe et affichage de son nom pour confirmer le bon dossier.
  • Présence du dossier game/tl.
  • Présence d’un dossier de traduction dans game/tl.
  • Présence de fichiers .rpy.
  • Détection de fichiers .rpy vides.
  • Détection des fichiers .json présents.
  • Détection possible d’un mauvais emplacement.
💡 Important : cet outil ne modifie aucun fichier. Il sert uniquement à aider à comprendre si la traduction est installée au bon endroit.
